À quoi correspondent les durées sur les boutons ?
La durée affichée = votre prochain intervalle si vous appuyez sur ce bouton
😓 Encore → 1 j (reset)
Vous n'avez pas su. La carte revient dans quelques cartes dans cette session, puis demain. L'intervalle repart de zéro.
🤔 Difficile → intervalle × 1,2
Vous avez hésité. L'intervalle croît lentement. Le facteur de facilité baisse légèrement.
😊 Bien → intervalle × facteur (~2,5×)
Bonne réponse. Multiplie l'intervalle par votre facteur de facilité personnel (démarre à 2,5 — s'adapte dans le temps).
🚀 Facile → intervalle × facteur × 1,3
Trop facile. L'intervalle est encore plus long et votre facteur de facilité augmente pour les prochaines fois.
💡 Pourquoi "1j / 1j" sur les deux premiers boutons ?
Pour une carte neuve ou en tout début d'apprentissage (interval = 0 ou 1), l'algorithme impose un minimum de 1 jour même sur "Difficile". C'est voulu : l'intervalle minimal évite de vous noyer en vous repassant la carte dans la journée quand vous avez d'autres cartes en attente.
Stockage des données
Où vivent vos cartes et statistiques
📦 localStorage du navigateur
Toutes vos données (cartes, intervalles, dates, historique) sont stockées localement sur votre appareil, dans le localStorage du navigateur. Aucun serveur, aucun compte.
PWA — navigateur (Chrome, Safari, Firefox)
→ localStorage du navigateur. Persistant tant que vous ne videz pas le cache ou les données du site.

PWA — installée sur écran d'accueil
→ Même localStorage que le navigateur d'installation (Chrome sur Android, Safari sur iOS). Les données survivent aux mises à jour.

TWA — Trusted Web Activity (app Android Play Store)
→ Le TWA génère un WebView Chrome dédié avec son propre localStorage isolé. Les données sont séparées du navigateur Chrome normal. Elles persistent après mise à jour de l'app mais sont effacées si l'utilisateur désinstalle ou vide le stockage de l'app dans les paramètres Android.

⚠️ Dans tous les cas : pensez à exporter régulièrement une sauvegarde globale.
